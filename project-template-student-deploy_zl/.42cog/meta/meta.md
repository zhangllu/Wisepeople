# 项目元信息 (Meta)

## 基本信息

**项目名称**: 智者网（WisePeople）

**项目标语**: 为成年人打造的通识教育地图

**版本**: 0.1.0

**创建日期**: 2026-04-27

**负责人**: 随机漫步

## 项目描述

智者网（WisePeople）致力于为成年人构建一套理解世界、社会与自我的全局知识系统。第一版内容来自《聪明的阅读者》，作者阳志平老师通过科学计量学与经典目录学，严选人类历史上 420 位智者及其一千余本代表作。项目倡导"合而御之"的学习路径，鼓励阅读者从"知识消费者"转变为"智慧创作者"，建立安身立命的学问。

## 目标用户

- 通识教育学习者: 希望系统性构建跨学科知识框架，但不知从何入手的成年人
- 深度阅读爱好者: 不满足于碎片化信息，渴望直达思想源头的终身学习者

## 核心价值主张

1. 严选逻辑：基于斯坦福全球 Top 2% 科学家排名、各领域终身成就奖及两千年文明积淀筛选，确保入选智者经受住 20-100 年时间考验
2. 文理并蓄：打破学科壁垒，将认知科学、心灵哲学与中国古典文化深度融合
3. 创作者导向：倡导从"知识消费者"转变为"智慧创作者"，在人类文明长河中留下自己的作品

## 核心板块

- **十大问题导览**: 围绕"天地人"三大维度，解答关于世界、历史、时代、社会、组织、家庭、人性、身体、信仰及"元知识"的终极困惑
- **智者库**: 荟萃跨学科的顶尖人物，从先秦诸子到 21 世纪的科学巨擘
- **代表作索引**: 精准推荐文笔优美、影响深远的个人专著
- **最小限度书单**: 为初学者提供"50 本必读经典"，从笛卡尔的沉思到《自私的基因》

- 框架: Next.js 15

**后端**:
- 运行时: Bun / Node.js

- ORM: Drizzle ORM

- 托管: Vercel / EdgeOne Pages

- 认证: Better Auth
