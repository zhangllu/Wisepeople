## 改进标准

### 01-智者介绍.md 改进清单

#### 必需章节（对照模板）
- [ ] 标题格式：`## English Name｜生卒年`
- [ ] 引用块：`> "名言"`
- [ ] 他是谁（200-300字，生动描述）
- [ ] 他的思想（4-6个核心概念）
- [ ] 他的底色（个人特质）
- [ ] 他对今天的意义（当代价值）
- [ ] 主要著作（表格格式，含难度）
- [ ] 对话问题（5-10个）
- [ ] 资料来源标注

#### 可选章节
- [ ] 与其他智者对比
- [ ] 阅读建议

### 02-原始资料.md 改进清单

- [ ] 核心概念（学术定义）
- [ ] 原始来源链接
- [ ] 关键争议（如有）

#### 可选章节
- [ ] 与同时代人对比
