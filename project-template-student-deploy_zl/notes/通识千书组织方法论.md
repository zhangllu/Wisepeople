# 通识千书的组织方法论

## 核心问题：如何组织这一千余本书？

人类文明数千年积累，使得我们现在习惯"分而治之"的学习倾向。尤其在工业革命之后，西方学界形成用一个学科解决一类问题的治学倾向。但是中国经学与目录学传统并非如此——传统是先读目录书，后读基本书，再围绕基本书写注疏。每一本基本书，实际上在回答为人处世常常面临的那些大问题。

吸纳中西文明各自精华，"通识千书"采取 **"大问题" → "学科或领域" → "智者" → "代表作"** 的四级组织逻辑。

---

## 一、如何组织"大问题"？

何谓"大问题"？就是那些短短人生百年，上至帝王将相，下至平民百姓，每个人都会反复面对的大问题；就是那些人类历史上，智者们前赴后继，始终有人在探讨的大问题。"大问题"因为与每一个人的生存发展息息相关而"大"，"大问题"因为成为人类文明历史基本命题而"大"。

### 三分类法：天地人

为了便于记忆，按照中国传统哲学观念的"天地人"三分法来归类：

| 维度 | 核心追问 |
|------|---------|
| **天** | 我们如何理解自己所处的世界、历史与时代 |
| **地** | 我们如何理解自己所处的社会、组织与家庭 |
| **人** | 我们如何理解自己或他人的人性、身体与信仰 |

### 两类知识

在具体解答时，离不开两类知识：

1. **源知识**——人类文明发源之处。潺潺溪流，肆意大海，今日盛行的各大人类文明离不开十余本源头著作，即"元典：人类文明十三经"。这一节单独处理，作为"通识千书"的开篇。
2. **元知识**——"知识的知识"，人类文明积累的各项知识背后的知识是什么。它是所有大问题的前置问题。

因此，九大问题 + 知识的知识 = **十大问题**。

---

## 二、如何组织"学科或领域"？

对同一个大问题，多个学科或领域往往会有不同的解答。考虑到人类工作记忆广度的限制，将其约束成 **"总论" + "四个方向"** 共五个主题的结构：

| 层级 | 说明 | 时间考验 |
|------|------|---------|
| **总论** | 从整体层面回答"大问题" | 挑选经受住百年以上时间考验的智者 |
| **四个方向** | 从侧面回答"大问题"，每个方向涉及若干学科或领域 | 挑选经受住 20 年以上时间考验的智者 |

---

## 三、如何组织"智者"？

人类大脑习惯记忆"人"，而非"抽象概念"。尝试对比两种记忆脉络：

- 先记住 10 个人，再记住每个人提出的 10 个术语
- 直接记忆 100 个术语

显然，我们更容易通过人来记住术语。

任何一个学科或领域的历史都是由人写就的。将那些远超学科或领域平均水准以上的人称为"智者"，他们或是某个学科的创始人，或是某个领域的集大成者。"智者"用作品说话，直面人类文明基本命题。

在每个主题下，挑选 **7~10 位智者**（少数主题例外）。少数作者在多个学科或领域均有突出贡献，因此会出现多次。

### 智者挑选标准

1. **年龄**：出生于 1960 年之前，少数学科或领域例外。作者的成就、人品已经（或接近）盖棺论定。
2. **时间**：在总论中侧重挑选人品、言论经得住百年以上时间考验的智者；在具体方向上，侧重挑选言论经得住 20 年以上时间尺度考验的智者。
3. **职业**：以学者、作家为主，数十年如一日，专攻某个领域或方向，而非偶然客串。
4. **作品**：侧重适合作为大众通识读物的作品，而非专业性较强的作品。

### 智者库来源

- **传记**：长期维护的智者传记库（中文数千本 + 英文上千本）
- **科学计量学**：谷歌学术、WOS 数据库中 H 指数大于 100 的学者（约 4700 余人）；顶级期刊高被引学者分析；全球科学家排名（斯坦福大学约安尼迪斯教授的工作）
- **奖项**：各学科代表性奖项（诺贝尔奖、图灵奖、终身成就奖等）；各领域名人堂
- **关于书的书**：WorldCat 数据库、谷歌图书；数百本关于书的书（《大哲学家》《西方正典》《巨匠与杰作》等）；各国经典通识教育丛书与课程设计
- **个人藏书**：作者 20 余年数万册藏书、数百篇书评、成千上万条读书札记

## 四、如何组织"代表作"？

如果只读一位智者的一本书，出于翻译质量原因或者心境原因，可能会读不下去，导致错过大师而不自知。反过来，如果连续读一位智者的多本书，而这些书都是他的"代表作"，读不下去的概率就小多了。

**每位智者，挑选 1~5 本代表作**，如有自传或传记，另行备注。

### 选书标准

| 优先 | 原则 |
|------|------|
| 优先挑选 | 文笔优美流畅，大众可读的个人专著 |
| 优先挑选 | 被学术圈广泛引用的著作 |
| 优先挑选 | 在未来百年依然重要的著作（自然科学/社科/工程：具备知识新鲜感；哲学/文学：历史上经久不衰的经典） |
| 放弃 | 多人合著的书（少数例外） |
| 放弃 | 市场价值大但学术价值不大的畅销书 |

---

## 总览：四级组织结构

```
十大问题（天地人 + 元知识）
  ├── 每个大问题 → 总论 + 四个方向（共 5 个主题）
  │     ├── 每个主题 → 7~10 位智者
  │     │     └── 每位智者 → 1~5 本代表作
  │     └── 50 个主题 × 7~10 位智者 = 420 位智者
  └── 总计：420 位智者，一千余本代表作
```
