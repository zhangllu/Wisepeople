# 亚里士多德（Aristotle）原始研究资料

> 收集日期：2026年5月
> 用途：智者介绍页写作参考

---

## 一、基本信息

- 全名：Aristotle（希腊语：Ἀριστοτέλης）
- 生卒：公元前384年 — 前322年（62岁）
- 出生地：古希腊斯塔吉拉（Stageira，今希腊哈尔基季基）
- 家庭：父亲尼各马可是马其顿国王的御医
- 老师：柏拉图（在柏拉图学园学习约20年）
- 学生：亚历山大大帝（曾担任其家庭教师）
- 绰号："The Teacher"（教师）或"The Philosopher"（哲学家）

---

## 二、生平大事

| 年份 | 事件 |
|------|------|
| 公元前384年 | 出生于斯塔吉拉 |
| 公元前367年 | 来到雅典，进入柏拉图学园，时约17岁 |
| 公元前347年 | 柏拉图去世，离开雅典 |
| 公元前347-345年 | 在小亚细亚阿索斯停留，与柏拉图主义者合作 |
| 公元前345-343年 | 在莱斯沃斯岛进行生物学研究 |
| 公元前343年 | 受马其顿国王腓力二世邀请，成为13岁的亚历山大的老师 |
| 公元前335年 | 回到雅典，创办吕克昂学园（Lyceum） |
| 公元前323年 | 亚历山大大帝去世，雅典反马其顿情绪高涨 |
| 公元前322年 | 离开雅典，次年去世于优卑亚岛 |

## 四、核心概念

**四因说（Four Causes）**
解释事物存在和变化的原因：
1. 质料因（material cause）：事物由什么构成
2. 形式因（formal cause）：事物的形式或本质
3. 动力因（efficient cause）：事物产生的推动者
4. 目的因（final cause）：事物存在的目的

**中庸之道（Golden Mean）**
美德在于两个极端之间的中道。例如：
- 勇敢是怯懦与鲁莽之间的中道
- 慷慨是吝啬与挥霍之间的中道

**人是政治动物（Zoon Politikon）**
人天生是城邦的动物，需要通过政治生活实现幸福（eudaimonia）

**三段论（Syllogism）**
逻辑推理的基本形式：大前提 + 小前提 → 结论
例如：所有人都会死（大前提）+ 苏格拉底是人（小前提）→ 苏格拉底会死（结论）

**幸福（Eudaimonia）**
人生的最高目的是幸福，但幸福不是快乐，而是灵魂按照美德进行活动

**实体-属性（Substance-Accident）**
实体是独立存在的个体，属性是依附于实体存在的性质

**潜能与现实（Potentiality and Actuality）**
变化是从潜能到现实的实现过程

## 五、吕克昂学园

- 公元前335年创办，位于雅典郊外吕克昂（Lyceum，原为阿波罗神庙）
- 也称"逍遥学派"（Peripatetic School），因为亚里士多德喜欢边走边讲学
- 学园有图书馆、标本室、地图等研究设施
- 比柏拉图学园更重视经验研究
- 延续数百年，直到公元529年与柏拉图学园一同被关闭

## 六、与柏拉图的对比

| 维度 | 柏拉图 | 亚里士多德 |
|------|--------|-----------|
| 世界观 | 两个世界（理念世界+可感世界） | 一个世界（形式在事物之中） |
| 获得知识的方式 | 回忆灵魂见过的理念 | 通过经验观察和逻辑推理 |
| 政治理想 | 哲人王统治的理想国 | 混合政体（ polity），中产阶级执政 |
| 文学态度 | 诗人应被逐出理想国 | 诗有教育价值（悲剧的净化功能） |
| 研究方法 | 对话录、辩证法 | 论文、系统分析、经验观察 |
| 师生关系 | 亚里士多德是柏拉图的学生 | "吾爱吾师，吾更爱真理" |

## 八、关键影响

**对中世纪的影响**
- 阿拉伯世界：阿维森纳（Ibn Sina）、阿维罗伊（Ibn Rushd）的注释
- 拉丁西方：托马斯·阿奎那将亚里士多德哲学与基督教神学结合

**对科学的影响**
- 生物学：他的动物学观察被达尔文评价为"无与伦比"
- 逻辑学：他的逻辑体系直到19世纪才被突破
- 物理学：他的物理学说统治西方直到伽利略时代

**对哲学的影响**
- 伦理学：德性伦理学的复兴（麦金泰尔、纳斯鲍姆）
- 政治学：公民共和主义传统
- 形而上学：实体-属性框架影响整个西方哲学
