# 智者资料库

## 简介

本资料库收集了各位智者的详细资料，包括面向用户的介绍、供核对的原始资料、以及AI认知方式说明。

## 📁 目录结构说明

```
智者资料库/
├── 00-模板文件/                 # ⭐ 格式模板和规范
│   ├── 00-格式规范手册.md       # 总体格式规范
│   ├── 01-智者介绍模板.md       # 介绍文档模板
│   ├── 02-原始资料模板.md       # 原始资料模板
│   └── 03-认知方式模板.md       # 认知方式模板
│
├── [slug]/                      # 智者目录（按slug命名）
│   ├── 01-智者介绍.md           # 面向用户的详细展示
│   ├── 02-原始资料.md           # 供核对的数据和来源
│   ├── 03-认知方式.md           # AI行为准则和思维路径
│   └── 04-人格卡.md（可选）      # 个人特质与矛盾
│
├── README.md                    # 本文档
├── 格式检查脚本.sh              # 格式检查工具
├── 查看进度.sh                  # 进度统计工具
└── 移动总结报告.md              # 文件移动记录
```

## 目录结构

每位智者的资料包含四个部分：

1. **01-智者介绍.md** - 面向用户的详细展示
   - 他是谁
   - 核心思想
   - 个人特质
   - 当代意义
   - 主要著作
   - 对话问题
   - 与其他智者的对比

2. **02-原始资料.md** - 供核对的数据和来源
   - 基本信息
   - 生平大事
   - 主要著作
   - 核心概念
   - 原始来源链接
   - 学术争议

3. **03-认知方式.md** - AI行为准则和思维路径
   - 主要认知方式
   - 思维路径
   - 提问方式
   - 明确反对的东西
   - 对话示例

4. **04-人格卡.md**（可选）- 个人特质与矛盾

## 🎯 使用指南

### 添加新智者
1. 查阅 `00-模板文件/` 中的模板和规范
2. 创建智者目录：`mkdir [slug]`
3. 复制相应模板到新目录
4. 按模板填写内容
5. 运行格式检查脚本验证

### 审阅现有文档
1. 运行格式检查脚本
2. 查看内容审阅报告（如有）
3. 对照模板修改不符合规范的部分
4. 确保所有必需章节完整

### 文档更新
1. 在文档末尾添加更新记录
2. 更新版本号
3. 运行格式检查
4. 提交更改

## 认知方式分类

基于结构阅读框架，九种认知方式：

### 信息型
1. **思想实验** - 像哲学家一样思考
2. **符号思考** - 像数学家一样思考
3. **实验科学** - 像心理学家/生物学家一样思考
4. **计算模拟** - 像计算机科学家一样思考
5. **田野调查** - 像人类学家/社会学家一样思考

### 叙事型
6. **幽默叙事** - 像段子手/相声演员一样思考
7. **故事叙事** - 像小说家/编剧一样思考

### 美感型
8. **文采美感** - 像诗人/散文家一样思考
9. **视觉美感** - 像画家/设计师一样思考
