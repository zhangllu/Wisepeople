## 核心概念

| 概念 | 简要说明 |
|------|---------|
| 系统1 / 系统2 | 快思考（直觉、自动）vs 慢思考（分析、控制） |
| 前景理论（Prospect Theory） | 取代期望效用理论，描述真实决策的框架 |
| 损失厌恶（Loss Aversion） | 损失的心理影响大约是同等收益的2倍 |
| 启发式（Heuristics） | 人类做判断时使用的心理捷径 |
| 锚定效应（Anchoring） | 初始数字对后续判断的系统性影响 |
| 峰终定律（Peak-End Rule） | 记忆由最强烈的时刻和结束时刻决定 |

## 主要影响与传承

- **师从**：伯克利受让·皮亚杰认知发展理论的影响较大（博士论文关于注意的语义分析）
- **影响**：行为经济学（塞勒、桑斯坦）、行为金融学、公共政策（助推理论）、医疗决策、管理决策
- **合作者**：阿莫斯·特沃斯基（最重要）、杰克·尼奇（早期合作）、芭芭拉·弗雷德里克森（后期幸福感研究）

## 关联智者

- **阿莫斯·特沃斯基**：卡尼曼最重要的合作者，没有之一。两人性格互补，智力对等，共同开创了行为经济学
- **赫伯特·西蒙**：西蒙的有限理性是卡尼曼理论的直接先驱——卡尼曼将西蒙的"理性是有限的"变成了"理性是有偏差的"
- **史蒂芬·平克**：两人立场互补——卡尼曼研究理性如何出问题，平克研究理性如何可能
- **理查德·塞勒**：将卡尼曼的研究应用于经济政策，是行为经济学的第二代旗手
- **基思·斯坦诺维奇**：发展了"三重心智模型"，超越卡尼曼的系统1/系统2框架
