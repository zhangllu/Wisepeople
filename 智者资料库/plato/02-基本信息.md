# 柏拉图（Plato）原始研究资料

> 收集日期：2026年5月
> 用途：智者介绍页写作参考

---

## 一、基本信息（来源：Wikipedia / Stanford Encyclopedia of Philosophy）

- 全名：Plato（希腊语：Πλάτων）
- 生卒：约公元前427年 — 前347年（约80岁）
- 出生地：古希腊雅典
- 本名：Aristocles，"Plato"是绰号（意为"宽广"，可能指他的宽阔肩膀或额头）
- 家庭：父亲Ariston，母亲Perictione，均来自雅典贵族家庭
- 老师：苏格拉底
- 学生：亚里士多德

## 二、生平大事

| 年份 | 事件 |
|------|------|
| 约公元前427年 | 出生于雅典贵族家庭 |
| 约公元前407年 | 成为苏格拉底的学生，时约20岁 |
| 公元前399年 | 苏格拉底被雅典民主政府判处死刑并饮毒酒身亡 |
| 公元前399-387年 | 离开雅典，游历埃及、昔兰尼加、大希腊（意大利南部） |
| 公元前387年 | 回到雅典，创办柏拉图学园（Academy），西方第一所大学 |
| 公元前367年 | 应邀赴叙拉古，教育年轻的狄奥尼修斯二世（失败） |
| 公元前361年 | 再次赴叙拉古（再次失败） |
| 约公元前347年 | 在雅典去世，约80岁 |

## 三、主要著作（均以对话录形式）

### 早期对话（苏格拉底时期）
- 《申辩篇》（Apology）：苏格拉底的辩护
- 《克里托篇》（Crito）：苏格拉底为何不越狱
- 《斐多篇》（Phaedo）：苏格拉底之死与灵魂不朽
- 《欧绪弗洛篇》（Euthyphro）：什么是虔诚？
- 《卡尔米德篇》（Charmides）：什么是节制？
- 《拉凯斯篇》（Laches）：什么是勇敢？

### 中期对话（柏拉图自己的思想开始成熟）
- 《高尔吉亚篇》（Gorgias）：修辞术与正义
- 《美诺篇》（Meno）：美德可教吗？回忆说
- 《斐德罗篇》（Phaedrus）：爱欲、修辞、灵魂不朽
- 《会饮篇》（Symposium）：什么是爱？爱欲的上升阶梯
- 《理想国》（Republic）：什么是正义？哲人王、洞穴寓言、理念论
- 《斐德罗篇》（Phaedrus）：美、爱、灵魂不朽

### 晚期对话（对自己理论的批判与修正）
- 《泰阿泰德篇》（Theaetetus）：什么是知识？
- 《巴门尼德篇》（Parmenides）：理念论的自我批判
- 《智者篇》（Sophist）：虚假陈述如何可能？
- 《政治家篇》（Statesman）：真正的政治技艺
- 《法篇》（Laws）：柏拉图最后一部作品，对《理想国》的修正
- 《蒂迈欧篇》（Timaeus）：宇宙创生论

## 四、核心概念

**理念论（Theory of Forms）**
可感世界只是理念世界的投影。理念（Forms）是永恒、不变、完美的真实存在。洞穴寓言：囚徒看见墙上的影子，以为那是真实，其实真正的世界在外面。

**理想国（The Republic）**
由"哲人王"统治的理想城邦。灵魂三分说：理性、激情、欲望对应统治者、军人、生产者三个阶层。

**灵魂不朽（Immortality of the Soul）**
灵魂先于身体存在，在身体死后继续存在。学习不是获取新知识，而是回忆（anamnesis）灵魂曾经见过的理念。

**爱欲（Eros）**
《会饮篇》中，爱是一种向上的运动，从爱一个美的身体，到爱所有美的身体，到爱美的灵魂，到爱美的知识，最后到达"美本身"。

**洞穴寓言（Allegory of the Cave）**
《理想国》第七卷。教育的本质是灵魂的转向——从变化的可感世界转向永恒的理念世界。

**回忆说（Anamnesis）**
《美诺篇》。学习即回忆。灵魂在进入身体前曾见过理念，学习是回忆那些知识。

## 五、柏拉图学园

- 成立于公元前387年，位于雅典郊外Academus树林
- 门口铭文："不懂几何者不得入内"
- 持续约900年（公元529年被查士丁尼大帝关闭）
- 培养了亚里士多德等众多思想家
- 是西方大学传统的源头

## 七、关键争议

**柏拉图是否"反民主"？**
《理想国》明确批评雅典民主制，认为民主会导致暴民统治。但有学者认为，柏拉图批评的不是民主本身，而是雅典当时的民主实践。

**理念论是柏拉图的原创吗？**
毕达哥拉斯学派已有类似思想。苏格拉底可能也在追问"美本身""正义本身"。但柏拉图是第一个系统阐述理念论的人。

**《书信》是否真作？**
现有十三封归于柏拉图名下的书信，真伪有争议。第七封信被认为是真作，对理解柏拉图的政治实践很重要。
