## 核心概念

| 概念 | 简要说明 |
|------|---------|
| 个人建构（Personal Construct） | 个人用来解释世界的一对维度（如"好 vs 坏"） |
| 建构系统（Construct System） | 个人拥有的全部建构的有机组织 |
| 建构选择论（Constructive Alternativism） | 任何事件都存在不止一种有效的解释方式 |
| 人是科学家（Man-the-Scientist） | 每个人都在提出假设、检验假设、修正理论 |
| 角色建构库（Repertory Grid / RepGrid） | 测量个人建构系统的工具 |
| 固定角色疗法（Fixed Role Therapy） | 通过扮演新角色来改变建构系统的治疗技术 |

## 主要影响与传承

- **师从**：在爱丁堡大学受戈弗雷·汤姆森（统计学家）影响，在爱荷华大学受卡尔·西肖尔影响
- **影响**：叙事心理学、建构主义心理治疗、认知行为疗法（CBT）、组织行为学、市场研究
- **传承者**：Fay Fransella（伦敦大学，个人建构心理学的主要推广者）、Don Bannister（英国心理学家）

## 关联智者

- **赫伯特·西蒙**：都关注认知结构，但西蒙专注认知的"局限"，凯利专注认知的"可能性"
- **卡尔·罗杰斯**：都反对精神分析的"病理化"倾向，强调人的主动性和自我成长
- **弗洛伊德**：凯利的理论是对弗洛伊德决定论的根本挑战——人不是被过去决定的，是被建构系统决定的
- **斯金纳**：行为主义说环境决定行为，凯利说对环境的解释决定行为——两种决定论的正面冲突
- **詹姆斯·马奇**：马奇的"身份驱动决策"与凯利的"建构驱动认知"有深刻的对称性
